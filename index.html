How It All Started

Three years ago, I was exactly where you might be — unsure if tech would ever make room for someone like me. I built Code with Serah to answer that question with a resounding yes, creating pathways that feel personal, human, and full of wonder.

From one screen and a stubborn idea, we have grown into a movement powered by experimentation, mentorship, and heart-driven STEM journeys.
